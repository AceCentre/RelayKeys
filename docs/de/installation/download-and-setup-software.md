# Download and Setup (Software)

Laden Sie das Installationsprogramm von [hier herunter](https://github.com/AceCentre/RelayKeys/releases/latest). Nach dem Herunterladen 'Ausführen' des Programms

Führen Sie den Installationsvorgang durch. Wählen Sie "Normale Installation" und lassen Sie die RelayKeys-Setup-Software ihre Arbeit tun, um die Software an der richtigen Stelle zu installieren (Standardmäßig: `C:\Program Files (x86)\Ace Centre\RelayKeys`).

{% hint style="info" %}
Wenn Sie "Portable" verwenden oder den Speicherort der Software ändern: Notieren Sie sich, wo Sie das Programm gespeichert haben, da Sie dies benötigen, wenn Sie es mit Ihrer Software verknüpfen
{% endhint %}
